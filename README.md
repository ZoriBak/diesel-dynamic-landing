
# DIESEL — SPILL THE REAL

A chaotic, AI-powered landing page inspired by Diesel's raw identity.

## Features
- Framer Motion animations
- AI brand voice (OpenAI)
- Webcam gesture interaction (basic MediaPipe-ready scaffold)
- Three.js style distortion background
- Fully deployable on Vercel

## Setup
1. npm install
2. Add OPENAI_API_KEY to .env.local
3. npm run dev

## Deploy
Push to GitHub → Import into Vercel
